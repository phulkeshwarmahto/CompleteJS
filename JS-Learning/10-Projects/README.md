# JavaScript Projects

This folder contains practical JavaScript projects to apply the concepts learned.

## Projects Included

1. **Rock Paper Scissors** - Game logic with DOM manipulation and events
2. **Tic Tac Toe** - Game state management and win detection
3. **Currency Converter** - API integration (optional) and form handling
4. **Dark Mode Toggle** - DOM and event handling

## How to Use

Each project typically includes:
- `index.html` - HTML structure
- CSS file - Styling
- `.js` file - JavaScript logic

Open the HTML file in your browser to run the project.

## Learning Points

- DOM manipulation and accessing elements
- Event handling (click, mouseover, etc.)
- State management
- Conditional logic
- Array methods and loops
